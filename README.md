# algoritme
In het kort gaat het algoritme alsvolgt (dijkstra):
1 kijk naar mogelijke stappen
2 pak kies de kortste stap (ook met de afgelegde stappen erbij)
3 ga verder bij 1.
4 als het eindpunt is behaald:
4.1 kortste route is gevonden
